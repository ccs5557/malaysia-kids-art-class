# Malaysia Kids Art Class Landing Page

A warm, mobile-first landing page for a Malaysian children's art class, built with React, Vite, TypeScript, and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is generated in `dist/`.

## Cloudflare Pages

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `20` or `22`

This project also includes Cloudflare Pages Functions for:

- `/robots.txt`
- `/sitemap.xml`
